Project:GoldMine → Zoho CRM Migration Example
👉 This is a working ETL demo project (ready to run with sample data)

Description

This repository demonstrates a basic ETL workflow for migrating CRM data from GoldMine exports to Zoho CRM.

Workflow

Export contacts from GoldMine as CSV
Transform field mappings
Load contacts via Zoho CRM API

Tools Used

Python
Zoho CRM REST API
CSV export from GoldMine
